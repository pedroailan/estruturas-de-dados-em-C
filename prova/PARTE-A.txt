	i(n) = n log n
	
	Existe um c e um n tal que f(n) >= c i(n) para qualquer n >= 0

	Sendo assim,
	
	n² + n + 10 >= n log n
  
       n + 1 + 10/n >= log (n)

		  n >= log (n) - 10/n - 1
